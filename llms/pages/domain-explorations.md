## Summary

Domain Explorations is a parked exploratory page. It describes the eventual shape for worked schemas, public-standard mappings, narrow workflow packs, and community reference builds, but it is not a current ontology catalog.

## Use This Page For

- Understanding why domain packs are not part of the current supported path.
- Explaining that domain examples need sharper open boundaries before promotion.
- Avoiding claims that Vellis ships finished regulated workflow packs today.

## Key Claims

- Domain packs need a sharper open boundary.
- A useful version should start from public standards or narrow workflows.
- The current supported path is the public Vellis repository, its local quickstart, and the documented beta walkthrough.
