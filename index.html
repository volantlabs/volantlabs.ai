Vellis · Apache-licensed graph engine

One memory your agents share — and you own.

Vellis is an Apache-licensed typed graph engine that turns scattered context into a living model your AI can reason over. Model your domain once; let Claude, Codex, and MCP-capable tools build on the same graph.

  • Clone the repo and run the graph engine locally
  • Connect Claude, Codex, and other MCP-capable tools through your own setup
  • Model schemas, provenance, and decisions as a typed graph
  • Read, traverse, write, propose, validate, and export
